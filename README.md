# 🛡️ **Knight in the dungeon** ⚔️
2D Souls-like with top-down view.

Game for KOD23 "Small, but mighty" game jam [https://gamin.me/compos/kod/23](https://gamin.me/compos/kod/23)

* WASD / arrows - move
* Mouse - rotation
* LMB - sword strike
* Space / RMB - rolling
* Esc - menu
* F11 - fullscreen/window mode
